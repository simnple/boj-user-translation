별을 품은 괄호 쌍

문제

올바른 괄호 문자열을 아래와 같이 정의해 보자.

  • 빈 문자열은 올바른 괄호 문자열이다.
  • 두 올바른 괄호 문자열을 이어붙인 문자열은 올바른 괄호 문자열이다.
  • $T$가 올바른 괄호 문자열이라면, ($T$)도 올바른 괄호 문자열이다.

예를 들어, ()(), (()())는 올바른 괄호 문자열이지만, )(, )()(()는 올바른 괄호 문자열이 아니다.

당신 앞에 올바른 괄호 문자열에 * 문자를 하나 추가한 문자열 $S$가 놓여 있다. $S$에 별을 품은 괄호 쌍이 몇 개 있을까?

  • $S_{l+1}, S_{l+2}, \cdots, S_{r-1}$를 순서대로 이어붙여 만든 문자열에 *이 포함되고, *을 제외한 문자열이 올바른 괄호 문자열이다.

입력

첫째 줄에 문자열 $S$가 주어진다. ($1 \le |S| \le 100$)
